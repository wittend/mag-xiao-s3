mag-xiao-s3 — XIAO ESP32S3 Magnetometer Node
=============================================

A small, battery/solar-powered firmware for the Seeed Studio XIAO ESP32S3 that samples a PNI RM3100
3‑axis magnetometer and an MCP9808 temperature sensor at 1 Hz, uses GNSS PPS for timing, and can
transmit data via LoRa.
